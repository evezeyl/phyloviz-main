The Aligned Sequences (Without explicit ID) Dataset type was chosen.

The data format for the typing data should be as follows. Load a text file containing only the sequences. All the sequences should have the same length. A unique sequential ID identifier will be attributed to each sequence based on the order of the sequences in the file. This format can also be used to analyze SNP data.

An example of the file content is the following: 
                                              
    ATATTTGTATATATGGCCCGAT
    ATATTTGATTATATGGCCCGAT
    ATATTTATTTATATGGCCCGAT
    ATATTAGTTTATATGGCCCGAT
    ATATATGTTTATATGGCCCGAT
    ATATTTGTTTATATGGCCCGAT
    ATTTTTGTTTATATGGCCCGAT
