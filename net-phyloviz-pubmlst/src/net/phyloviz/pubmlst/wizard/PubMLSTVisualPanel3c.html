Choose this option if you have data on a file on your computer that you want to use with the downloaded allelic profiles.

Press Browse and select the file. This file should be a tab separated file with the first row with column headers (field names).
One of the fields should be the Sequence identifier. Choose this field in the Key drop down menu to link the ancillary data to the allelic profile data. If the field does not appear in the drop-down list press the Update button.

Finally press Finish to create the dataset.
