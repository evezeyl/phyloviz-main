This plugin creates a PHYLOViZ dataset by downloading allelic profile and isolate data from PubMLST.org databases, using the available webservices.

An internet connection is necessary to communicate with the PubMLST.org webservice.

Choose a dataset name and an available online database for a given microorganism to continue.

Warning: the download of a dataset and isolate data can take considerable time depending on the number of STs and isolates available.

This is a BETA version of the plugin. Please contact us if some bug is detected.
