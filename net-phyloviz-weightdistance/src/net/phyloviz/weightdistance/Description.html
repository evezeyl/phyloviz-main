This distance adds a first tiebreak rule to the goeBURST rules based on the Euclidean distance between two profiles. It assumes that the genetic distance between two MLVA profiles is proportional to the difference in the number of repeats at each locus. NEW

For goeBURST algorithm, it will still link only two Sequence Types only if they are SLVs.

The complete list of rules used to create the trees are:
  • 1: Euclidean distance between profiles
  • 2: #SLVs
  • 3: #DLVs
  • 4: #TLVs
  • 5 to (n+1): #(4 to n)LVs
  • n+2: Frequency of an ST
  • n+3: STID
